अर्जुनविशादयोग

श्लोक १.१

धृतराष्ट्र उवाच
धर्मक्षेत्रे कुरुक्षेत्रे समवेता युयुत्सवः ।
मामकाः पन्डवाश्चैव किमकर्वत सन्जय ॥१.१॥

टीक / टिप्पणि

श्लोक १.२

सञ्जय ऊवाच
दृष्ट्वा तु पाण्डवानीकं व्यूढं दुर्योधनस्तदा।
आचार्यमुपसङ्गम्य राजा वचनमब्रवीत् ॥१.२॥

टीक / टिप्पणि

श्लोक १.३

पश्यैतां पाण्डुपुत्राणामाचार्य महतीं चमूम्।
व्यूढां द्रुपदपुत्रेण तव शिष्येण धीमता ॥१.३॥

टीक / टिप्पणि

श्लोक १.४

अत्र शूरा महेष्वासा भीमार्जुनसमा युधि।
युयुधानो विराटश्च द्रुपदश्च महारथः ॥१.४॥

टीक / टिप्पणि

श्लोक १.५

धृष्टकेतुश्चेकितानः काशिराजश्च वीर्यवान्।
पुरुजित्कुन्तिभोजश्च शैब्यश्च नरपुङ्गवः ॥१.५॥

टीक / टिप्पणि

श्लोक १.६

युधामन्युश्च विक्रान्त उत्तमौजाश्च वीर्यवान्।
सौभद्रो द्रौपदेयाश्च सर्व एव महारथाः ॥१.६॥

टीक / टिप्पणि

श्लोक १.७

अस्माकं तु विशिष्टा ये तान्निबोध द्विजोत्तम।
नायका मम सैन्यस्य संज्ञार्थं तान्ब्रवीमि ते ॥१.७॥

टीक / टिप्पणि

श्लोक १.८

भवान्भीष्मश्च कर्णश्च कृपश्च समितिञ्जयः।
अश्वत्थामा विकर्णश्च सौमदत्तिस्तथैव च ॥२.८॥

टीक / टिप्पणि

श्लोक १.९

अन्ये च बहवः शूरा मदर्थे त्यक्तजीविताः।
नानाशस्त्रप्रहरणाः सर्वे युद्धविशारदाः ॥१.९॥

टीक / टिप्पणि

श्लोक १.१०

अपर्याप्तं तदस्माकं बलं भीष्माभिरक्षितम्।
पर्याप्तं त्विदमेतेषां बलं भीमाभिरक्षितम् ॥१.१०॥

टीक / टिप्पणि

श्लोक १.११

अयनेषु च सर्वेषु यथाभागमवस्थिताः।
भीष्ममेवाभिरक्षन्तु भवन्तः सर्व एव हि ॥१.११॥

टीक / टिप्पणि

श्लोक १.१२

तस्य संजनयन्हर्षं कुरुवृद्धः पितामहः।
सिंहनादं विनद्योच्चैः शङ्खं दध्मौ प्रतापवान् ॥१.१२॥

टीक / टिप्पणि

श्लोक १.१३

ततः शङ्खाश्च भेर्यश्च पणवानकगोमुखाः।
सहसैवाभ्यहन्यन्त स शब्दस्तुमुलोऽभवत् ॥१.१३॥

टीक / टिप्पणि

श्लोक १.१४

ततः श्वेतैर्हयैर्युक्ते महति स्यन्दने स्थितौ।
माधवः पाण्डवश्चैव दिव्यौ शङ्खौ प्रदध्मतुः ॥१.१४॥

टीक / टिप्पणि

श्लोक १.१५

पाञ्चजन्यं हृषीकेशो देवदत्तं धनंजयः।
पौण्ड्रं दध्मौ महाशङ्खं भीमकर्मा वृकोदरः ॥१.१५॥

टीक / टिप्पणि

श्लोक १.१६

अनन्तविजयं राजा कुन्तीपुत्रो युधिष्ठिरः।
नकुलः सहदेवश्च सुघोषमणिपुष्पकौ ॥१.१६॥

टीक / टिप्पणि

श्लोक १.१७

काश्यश्च परमेष्वासः शिखण्डी च महारथः।
धृष्टद्युम्नो विराटश्च सात्यकिश्चापराजितः ॥१.१७॥

टीक / टिप्पणि

श्लोक १.१८

द्रुपदो द्रौपदेयाश्च सर्वशः पृथिवीपते।
सौभद्रश्च महाबाहुः शङ्खान्दध्मुः पृथक्पृथक् ॥१.१८॥

टीक / टिप्पणि

श्लोक १.१९

सघोषो धार्तराष्ट्राणां हृदयानि व्यदारयत्।
नभश्च पृथिवीं चैव तुमुलो व्यनुनादयन् ॥१.१९॥

टीक / टिप्पणि

श्लोक १.२०

अथ व्यवस्थितान् दृष्ट्वा धार्तराष्ट्रान्कपिध्वजः।
प्रवृत्ते शस्त्रसंपाते धनुरुद्यम्य पाण्डवः ॥१.२०॥

टीक / टिप्पणि

श्लोक १.२१

अर्जुन उवाच
हृषीकेशं तदा वाक्यमिदमाह महीपते।
सेनयोरुभयोर्मध्ये रथं स्थापय मेऽच्युत ॥१.२१॥

टीक / टिप्पणि

श्लोक १.२२

यावदेतान्निरीक्षेऽहं योद्धुकामानवस्थितान्।
कैर्मया सह योद्धव्यमस्मिन्रणसमुद्यमे ॥१.२२॥

टीक / टिप्पणि

श्लोक १.२३

योत्स्यमानानवेक्षेऽहं य एतेऽत्र समागताः।
धार्तराष्ट्रस्य दुर्बुद्धेर्युद्धे प्रियचिकीर्षवः ॥१.२३॥

टीक / टिप्पणि

श्लोक १.२४

संजय उवाच
एवमुक्तो हृषीकेशो गुडाकेशेन भारत।
सेनयोरुभयोर्मध्ये स्थापयित्वा रथोत्तमम् ॥१.१४॥

टीक / टिप्पणि

श्लोक १.२५

भीष्मद्रोणप्रमुखतः सर्वेषां च महीक्षिताम्।
उवाच पार्थ पश्यैतान्समवेतान्कुरूनिति ॥१.२५॥

टीक / टिप्पणि

श्लोक १.२६

तत्रापश्यत्स्थितान्पार्थः पितॄनथ पितामहान्।
आचार्यान्मातुलान्भ्रातृ़न्पुत्रान्पौत्रान्सखींस्तथा ॥१.२६॥

टीक / टिप्पणि

श्लोक १.२७

श्वशुरान्सुहृदश्चैव सेनयोरुभयोरपि।
तान्समीक्ष्य स कौन्तेयः सर्वान्बन्धूनवस्थितान् ॥१.२७॥

टीक / टिप्पणि

श्लोक १.२८

कृपया परयाऽऽविष्टो विषीदन्निदमब्रवीत्।

अर्जुन उवाच
दृष्ट्वेमं स्वजनं कृष्ण युयुत्सुं समुपस्थितम् ॥१.२८॥

टीक / टिप्पणि

श्लोक १.२९

सीदन्ति मम गात्राणि मुखं च परिशुष्यति।
वेपथुश्च शरीरे मे रोमहर्षश्च जायते ॥१.२९॥

टीक / टिप्पणि

श्लोक १.३०

गाण्डीवं स्रंसते हस्तात्त्वक्चैव परिदह्यते।
न च शक्नोम्यवस्थातुं भ्रमतीव च मे मनः ॥१.३०॥

टीक / टिप्पणि

श्लोक १.३१

निमित्तानि च पश्यामि विपरीतानि केशव।
न च श्रेयोऽनुपश्यामि हत्वा स्वजनमाहवे॥१.३१॥

टीक / टिप्पणि

श्लोक १.३२

न काङ्क्षे विजयं कृष्ण न च राज्यं सुखानि च।
किं नो राज्येन गोविन्द किं भोगैर्जीवितेन वा॥१.३२॥

टीक / टिप्पणि

श्लोक १.३३

येषामर्थे काङ्क्षितं नो राज्यं भोगाः सुखानि च।
त इमेऽवस्थिता युद्धे प्राणांस्त्यक्त्वा धनानि च॥१.३३॥

टीक / टिप्पणि

श्लोक १.३४

आचार्याः पितरः पुत्रास्तथैव च पितामहाः।
मातुलाः श्चशुराः पौत्राः श्यालाः सम्बन्धिनस्तथा॥१.३४॥

टीक / टिप्पणि

श्लोक १.३५

एतान्न हन्तुमिच्छामि घ्नतोऽपि मधुसूदन।
अपि त्रैलोक्यराज्यस्य हेतोः किं नु महीकृते॥१.३५॥

टीक / टिप्पणि

श्लोक १.३६

निहत्य धार्तराष्ट्रान्नः का प्रीतिः स्याज्जनार्दन।
पापमेवाश्रयेदस्मान्हत्वैतानाततायिनः॥१.३६॥

टीक / टिप्पणि

श्लोक १.३७

तस्मान्नार्हा वयं हन्तुं धार्तराष्ट्रान्स्वबान्धवान्।
स्वजनं हि कथं हत्वा सुखिनः स्याम माधव॥१.३७॥

टीक / टिप्पणि

श्लोक १.३८

यद्यप्येते न पश्यन्ति लोभोपहतचेतसः।
कुलक्षयकृतं दोषं मित्रद्रोहे च पातकम्॥१.३८॥

टीक / टिप्पणि

श्लोक १.३९

कथं न ज्ञेयमस्माभिः पापादस्मान्निवर्तितुम्।
कुलक्षयकृतं दोषं प्रपश्यद्भिर्जनार्दन॥१.३९॥

टीक / टिप्पणि

श्लोक १.४०

कुलक्षये प्रणश्यन्ति कुलधर्माः सनातनाः।
धर्मे नष्टे कुलं कृत्स्नमधर्मोऽभिभवत्युत॥१.४०॥

टीक / टिप्पणि

श्लोक १.४१

अधर्माभिभवात्कृष्ण प्रदुष्यन्ति कुलस्त्रियः।
स्त्रीषु दुष्टासु वार्ष्णेय जायते वर्णसङ्करः॥१.४१॥

टीक / टिप्पणि

श्लोक १.४२

सङ्करो नरकायैव कुलघ्नानां कुलस्य च।
पतन्ति पितरो ह्येषां लुप्तपिण्डोदकक्रियाः॥१.४२॥

टीक / टिप्पणि

श्लोक १.४३

दोषैरेतैः कुलघ्नानां वर्णसङ्करकारकैः।
उत्साद्यन्ते जातिधर्माः कुलधर्माश्च शाश्वताः॥१.४३॥

टीक / टिप्पणि

श्लोक १.४४

उत्सन्नकुलधर्माणां मनुष्याणां जनार्दन।
नरकेऽनियतं वासो भवतीत्यनुशुश्रुम॥१.४४॥

टीक / टिप्पणि

श्लोक १.४५

अहो बत महत्पापं कर्तुं व्यवसिता वयम्।
यद्राज्यसुखलोभेन हन्तुं स्वजनमुद्यताः॥१.४५॥

टीक / टिप्पणि

श्लोक १.४६

यदि मामप्रतीकारमशस्त्रं शस्त्रपाणयः।
धार्तराष्ट्रा रणे हन्युस्तन्मे क्षेमतरं भवेत्॥१.४६॥

टीक / टिप्पणि

श्लोक १.४७

सञ्जय उवाच
एवमुक्त्वाऽर्जुनः संख्ये रथोपस्थ उपाविशत्।
विसृज्य सशरं चापं शोकसंविग्नमानसः॥१.४७॥

टीक / टिप्पणि


॥ ॐ तत्सदिति श्रीमद्भगवद्गीतासूपनिषत्सु ब्रह्मविद्यायां योगशास्त्रे श्रीकृष्णार्जुनसंवादे सांख्ययोगो नाम प्रथमोऽध्यायः ॥ १ ॥
